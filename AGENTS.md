# Инструкции для Codex

Ты работаешь над проектом "Мини-CRM товаров v1".

Главная цель:
Сделать простую товарную админку для малого бизнеса.

Это НЕ полноценная CRM продаж.

Что входит:
- авторизация
- dashboard
- товары
- карточка товара
- фото и видео
- категории
- пользовательские поля
- ключевые слова
- импорт Excel/CSV
- read-only API для ИИ
- Telegram-добавление товара позже

Главное архитектурное правило:
Все бизнес-таблицы должны иметь company_id.

Дизайн:
- светлая SaaS-панель
- Tailwind CSS
- shadcn/ui
- синий акцент
- белые карточки
- мягкие тени
- аккуратные таблицы
- без перегруза

Не добавлять в MVP:
- оплату
- воронку продаж
- чаты
- обработку заявок AI
- маркетплейсы
- 1С
- сложную аналитику
- мобильное приложение

Работать поэтапно.
Не писать весь проект сразу.
После каждого этапа проверять TypeScript, lint и build.

Ты senior fullstack-разработчик и системный архитектор.
Мне нужно разработать MVP web-системы: Мини-CRM товаров v1.
Это не полноценная CRM продаж. Это простая товарная админка для малого бизнеса, чтобы клиент
мог быстро оцифровать товары, фото, видео, цены, остатки, дополнительные поля и ключевые
слова.
Главная цель:
Клиент добавляет товары через web-кабинет, импорт Excel/CSV или Telegram-бота. Система
хранит каталог и отдаёт товары через read-only API для внешнего AI-бота.
Важно:
AI не обрабатывает заявки и не ведёт клиентов. AI используется только как помощник при
добавлении товара через Telegram: помогает красиво оформить описание, извлечь характеристики
и предложить ключевые слова.

Артикул:
Формат:
[COMPANY_PREFIX]-[CATEGORY_CODE]-[PRODUCT_CODE]
Пример:
JWL-001-A7K9
Где:
JWL - префикс компании
001 - код категории
A7K9 - уникальный случайный код товара
Пользовательские поля:
Клиент может сам создавать дополнительные поля товара.
Типы MVP:
- текст
- число
- список
- да/нет
Ключевые слова:
В карточке товара должно быть поле keywords.
Оно используется для поиска товара и API для AI.
Telegram flow:
1. Пользователь нажимает Добавить товар
2. Бот показывает инструкцию по правильной съемке фото/видео
3. Пользователь отправляет фото/видео
4. Система проверяет качество и обрабатывает медиа
5. Пользователь пишет описание товара как умеет
6. AI извлекает название, цену, остаток, категорию, характеристики
7. AI генерирует красивое описание и ключевые слова
8. Система генерирует артикул
9. Бот показывает предварительную карточку
10. Пользователь подтверждает
11. Товар сохраняется в базу
